## How to use this repo
1. Clone the repo
```
git clone https://git.garena.com/sea-labs-id/batch-01/bagus-tambunan/final-project-frontend.git
```
2. Open the dir on your terminal
```
cd final-project-frontend
```
3. Fetch dependencies
```
npm install
```
4. Start server:
```
npm start
```
